Circle Restaurant Management

Circle is a full-stack, well rounded restaurant management application. It allows users, such as customers, to add items to their order while also facilitating restaurant administrative functions. On the administrative side this would include additional features such as adding, editing and deleting existing orders. This full-stack application was created using Bob's Burgers as inspiration but can be changed to suit any and all restaurants.

Employee Tracker System

This application runs strictly through terminal after using Node.js. With business owners in mind, this employee tracker allows owners to easily manage departments and edit their employee rosters with ease. This command-line application utilizes a MySql database enabling features to add, delete, update any and all employee information.
